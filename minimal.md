# Welcome to Storier!
This engine now supports:

- Markdown headings  
- Paragraph text  
- **Bold inline formatting**  
- *Italic formatting*  
- [Clickable-looking links](https://example.com)  
- Code blocks  
- Scripting with Nim DSL in code blocks  

```nim on:render
drawText("Hello from DSL!", 40, 200, 22, "yellow")
```
